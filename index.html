// §04 Spot the Bug

A small taste of how a lesson runs : find the bug before you check the fix.

def factorial(n): if n == 0: return 0 # base case return n * factorial(n - 1)
The base case returns 0, so every call multiplies by zero somewhere in the chain — factorial(n) always evaluates to 0. The fix: the base case should return 1, since multiplying by 1 doesn't change the running product.
